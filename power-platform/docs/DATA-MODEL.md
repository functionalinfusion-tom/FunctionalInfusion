## Phase 2 Tables (planned)

### fi_inventorylot
Tracks received ingredient lots, remaining quantity, COA documents.
Key fields: fi_supplierlot, fi_internallotcode, fi_locationid, fi_receivedqty, fi_remainingqty, fi_status (Available/Quarantine/Consumed/Expired), fi_expirationdate, fi_coastatus

### fi_batchrecord
Production run records.
Key fields: fi_lotcode (generated by flow), fi_recipemasterid, fi_recipeversionid (snapshot — locked), fi_customerid, fi_batchdate, fi_batchsize, fi_batchunittype, fi_status (Planned/InProgress/PendingQA/Complete/OnHold), fi_crewlead, fi_tankline

### fi_batchingredientactual
Actual ingredient usage per batch linked to inventory lots.
Key fields: fi_batchrecordid, fi_ingredientid, fi_inventorylotid, fi_theoreticalqty (frozen), fi_actualqty (operator entry), fi_variance (calculated), fi_pickedby, fi_pickeddatetime, fi_scannedlotcode

### fi_qatest
QA test results per batch.
Key fields: fi_batchrecordid, fi_testtype (pH/Brix/Potency/Micro/etc), fi_expectedmin, fi_expectedmax, fi_actualresult, fi_passfail (calculated), fi_testedby, fi_testdatetime, fi_iscritical, fi_notes
